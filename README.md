# etheralley-web-interface

The web interface for the Ether Alley platform

## Local Setup

1. Download [node.js v16.x](https://nodejs.org/en/download/)

2. Install dependancies

   ```sh
   npm install
   ```

3. Serve the web app locally
   ```sh
   npm run start
   ```
